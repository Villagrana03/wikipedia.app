switch

Learn how to control code execution with switch statements.

The switch statement is a type of conditional statement in C++ just like if and else. A switch statement provides a means of checking an expression against various cases. If there is a match, the code within the block starts to execute.

switch Syntax

switch (grade) {
--- case 9:
------ std::cout << "Freshman\n";
------ break;
--- case 10:
------ std::cout << "Sophomore\n";
------ break;
--- case 11:
------ std::cout << "Junior\n";
------ break;
--- case 12:
------ std::cout << "Senior\n";
------ break;
--- default:
------ std::cout << "Invalid\n";
------ break;
}

Note: The expression of a switch statement must be an integral type (int, char, short, long, long long, or enum).

switch vs. if-else

The break Keyword

The break keyword signals the compiler to jump out of a case. Without the break keyword at the end of each case, the program would execute the code for the first matching case and also subsequent cases.

For example, if the break keyword is removed from the first case of the example:

case 9:
--- std::cout << "Freshman\n";
case 10:
---std::cout << "Sophomore\n";
break;

Output
Freshman
Sophomore

The default Keyword

The default keyword is used to specify a code block to be run if none of the cases are true. It acts very similarly to an else statement and is usually placed at the end of a switch statement.

default:
std::cout << "Invalid\n";
break;
