Loops

A loop is a control flow construct that repeatedly executes some code. Loops are great for tasks that have to repeat multiple times in a row.

I will show the following loops:

  • While Loop
  • do-while loop
  • for loop
  • for-each loop

While loop

The while loop repeats a block of code as long as the given boolean condition is true. For example:

int count = 1;
while (count <= 5) {
--- std::cout << count; //This is the print() in c++
--- count++;
}

Output => 12345

Infinite Loops

If the expression of a while loop always evaluates to true, the loop will keep repeating forever. This is called an infinite loop.

int count = 1;
while (count <= 5) {
--- std::cout << count; //This will be a infinite loop because it does not have a "counter" variable
}

do-while loop

The do-while loop is very similar to a while loop, but will always execute at least once, even if the condition is initially false:

//Even though price > 500 evaluates to false, the program above will still print Too expensive! once.
int price = 300;
do {
---std::cout << "Too expensive!";
} while (price > 500);
//Note: Unlike the while loop, a semicolon ; needs to be placed at the end of a do-while loop!

for Loops

The for loop should be implemented when the programmer knows exactly how many times a block of code needs to be repeated.

for (int i = 1; i <= 5; i++) {
--- std::cout << i;
}
Output => 12345

for-each Loops

The for-each loop is especially useful for iterating through list-like structures such as arrays or std::vector. Although regular for loops are perfectly capable of this task, for-each loops provides a more convenient syntax:

This for-each loop iterates through an array called fibonacci using the counter variable number.

int fibonacci[5] = {0, 1, 1, 2, 3};
for (int number : fibonacci) {
--- std::cout << number;
}

The auto Keyword

When iterating through an array using the for-each loop, the counter variable may be declared using the auto keyword.

auto lets C++ deduce the type of the variable based on the type of the list it belongs to.

int fibonacci[5] = {0, 1, 1, 2, 3};
for (auto number : fibonacci) {
--- std::cout << number;
}

break

The break keyword can be used to jump out of a loop. When the break statement is executed, the loop immediately ends and any code below it will not run:

break vs. return

A break statement is used to terminate a switch or loop, and code execution will continue to the line after the switch or loop.

A return statement terminates the entire function and resumes code execution at the point where the function was called. Unlike break, return also returns something back to the caller.

Continue

The continue keyword can be placed inside a loop to skip over one iteration of the loop. If continue is executed, the current loop iteration will immediately end, and the next iteration will begin.
