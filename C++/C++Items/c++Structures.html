c++ Structures

In C++, structures (commonly referred to as struct) are user-defined data types that allow you to group together variables of different types under a single name. They are a fundamental way to organize and manage related data in a program.

Key Features of struct:

  • Custom Data Types: A structure is essentially a blueprint for creating objects that hold related data.
  • Multiple Members: It can contain multiple members of different data types.
  • Access Specifiers: Members are public by default (unlike classes, where they are private).
  • Efficient Organization: Useful for logically grouping data (e.g., representing a person with name, age, and address).

Structures syntax

struct StructureName {
dataType1 member1;
dataType2 member2;
// More members
};


struct Person {
string name;
int age;
float height;
};
