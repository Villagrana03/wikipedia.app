Program Structure

Welcome to the basic syntax section of Learn C++ for Programmers! This section will cover concepts that are essential for writing your first C++ program.

Hello World

#include <iostream>
using namespace std;

int main() {
cout << "Hello, world!";
return 0;
}

Preprocessor Directive

IOstream

is a special type of statement that starts with the # symbol and is examined before actual compilation, The library being imported in this program is the iostream library, which stands for input/output stream. This library allows us to use cout on line 5 and print the hello world message.

#include <iostream>

using nameSpace

line introduces the namespace std into the current program. This line allows the usage of standard library objects without prepending the std:: identifier.

using namespace std;

main(){}

This is the beginning of the main function. Every C++ program must have a main function in order to run.

int main(){
cout << "Hello, world!"; //is responsible for printing out the hello world message.
return 0; //The next line is a return statement, which terminates a function and returns something to the caller. A return value of 0 indicates that the main function finished running with no problems.
}

COUT

std::cout, which stands for “character output”, is used together with the insertion operator << to print values on the terminal:

std::cout << "Hello World!"; Output Hello world!

New line

A new line can be inserted by adding "\n" in the text to be printed.

std::cout << "Hello World!\n";
std::cout << "C++ is fun!\n";

std::endl, which stands for “end line”, can also be used to achieve the same result:

std::cout << "Hello World!" << std::endl;
std::cout << "C++ is fun!" << std::endl;
