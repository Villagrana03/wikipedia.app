Variables

Variables

A variable is a name that represents a particular piece of the computer’s memory that has been set aside for storing, retrieving, and using data. This article will cover the following topics:

  • Declaring a variable
  • Declaring multiple variables
  • Constant variables

Creating a Variable

In order to create a variable, three pieces of information need to be present:

  1. Data type: The type of data the variable will store.
  2. Name: The unique name of the variable. Names cannot start with a number and cannot be a C++ keyword.
  3. Value: The value the variable will store. This value must match the type of the variable.


type name = Value;

int age = 21;
string name = "Michelle";


//When a a variable does not have value we just declare it with the data type and name
int money;

Declaring Multiple Variables

Multiple variables of the same type can be declared in a single statement using a comma-separated list. For example:

// Declare three integer variables
int a = 1, b = 2, c = 3;

Data Types

A variable in C++ must be assigned a specific type. Type is important because it tells the compiler how to interpret the content of a variable. There are 5 basic data types that every C++ programmer should know about:

  • int
  • double
  • char
  • string
  • bool

int

int stores whole numbers without decimals. An int variable requires 4 bytes of memory space and ranges from -2³¹ to 2³¹.

int moonLanding = 1969;
int age = 18;

double

double stores floating-point numbers with decimals. A double variable requires 8 bytes of memory space and is sufficient for storing up to 15 decimal digits.

double pi = 3.1415;
double height = 1.75;

Type Conversion

It is possible to convert int into double and vice versa. This is called a type conversion or a type casting.

The notation (type) value means “convert value to type“. So for example:

// Converting a double to an int
double a = 3.5;
int b = (int) a; // b is now 3

// Converting an int to a double
int c = 5;
double d = (double) c; // d is now 5.0

char

char stores a single character surrounded by single quotes ' '. A char variable requires 1 byte of memory space.

char first_letter = 'a';

string

std::string stores a sequence of characters surrounded by double quotes " ".

Note: std::string is not a built-in type; the <string>library must be included before std::string can be used.

#include <string>

std::string message = "Hello World!";

bool

bool stores true or false boolean values. A bool variable requires 1 byte of memory space.

bool underaged = true;
bool certified = false;
