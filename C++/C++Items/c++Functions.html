Functions

A function in C++ contains a set of instructions that are executed when it is called. Functions are an important part of any good C++ program because they make code flexible and reusable.

In this article, we will cover the following topics:

  • Creating a function
  • Calling a function
  • Return type and value
  • Function scope

Creating a function

While C++ provides many helpful built-in functions, it’s never a bad idea to create your own user-defined functions to tackle specific tasks. A function is made up of two components:

  1. Declaration: The function’s name, return type, and any parameters.
  2. Definition: The body of the function containing the code to be executed.

//void means that the function does not have a return value.
void doSomething() { // Declaration
--- std::cout << "Functions are great!\n"; // Definition
}

Calling a function

To use a function, it needs to be “called” by specifying the function name followed by a pair of parentheses () and a semicolon ;:

// Create a function
void doSomething() {
std::cout << "Functions are great!\n";
}

int main() {
// Call the function
--- doSomething();
}

Return Type and Value

The programmer can choose whether a function returns something back to the caller. A function that doesn’t return a value should be declared as a void function.

A function that returns a value need to have two things:

  • Return type: The type defined before the function’s name. Just like variables, functions can take on data types like int, double, and bool. This type dictates the type of value that can be returned.
  • Return value: The specific value returned from the function using the return keyword. The type of this return value must match the function’s return type.

double valueOfPi() {
return 3.1415;
}

int main() {
double pi = valueOfPi();
std::cout << pi; // Output: 3.1415
}

Function Scope

Scope defines the region of code that can access or view a given variable. Function scope is no different from other types of scopes; variables declared inside a function are only accessible inside that function. If we try to reference a function variable outside the function scope, the C++ compiler will output an error message.

int lucky() {
int lucky_number = 7;
return lucky_number;
}

int main() {
//Error: variable lucky_number is not in the scope of main()
std::cout << lucky_number;

return 0;
}

Parameters

In C++, function parameters are variables that act as placeholders for input values passed to a function. They act as variables inside a function.

int feetToInches(int ft) { //Here we have "ft" as a parameter
return ft * 12;
}

Function Overloading

In C++, function overloading allows multiple functions to have the same name as long as they differ in their parameters. It enables functions to handle different types/numbers of inputs.

// Adds two int values
--- int add(int a, int b) {
return a + b;
}

// Adds two double values
double add(double a, double b) {
--- return a + b;
}
