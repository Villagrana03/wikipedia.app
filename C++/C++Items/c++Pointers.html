Pointers

Pointers are a concept found primarily in languages like C and C++ that allow direct access to memory by "pointing" to the location where data is stored. A pointer is essentially a variable that holds the memory address of another variable, rather than holding an actual data value.

Key concepts of Pointers

  • Memory Address: Each piece of data in a program is stored in a specific location in memory, which has an address. A pointer variable holds this memory address.
  • Dereferencing: Dereferencing a pointer means accessing the value stored at the memory address it points to. In C and C++, this is done using the * operator.

Syntax and Basics

Must have the same variable type in order to access the address
int a = 5;
int *p = &a; // p is a pointer to int, holding the address of a
//& is the address-of operator, and * is the dereference operator.

*

  • * (Dereference Operator): Used to access or modify the value at the address stored in a pointer. For example, *ptr gives the value at the address stored in ptr.

&

  • & (Address-of Operator): Used to get the address of a variable. For example, &value gives the memory address where value is stored.

// Declare a pointer to an integer
int *ptr;

// Use the & operator to get the address of num and store it in ptr
ptr = & num; // ptr now holds the address of num

// Print the value of num directly
cout << "Value of num: " << num << endl; // Output: 10

// Print the address of num using the & operator
cout << "Address of num: " << &num << endl;

// Print the value stored in ptr (which is the address of num)
cout << "Value of ptr (Address of num): " << ptr << endl;

// Dereference ptr to get the value stored at the address it points to (i.e., the value of num)
cout << "Value at the address ptr is pointing to: " << *ptr << endl; // Output: 10

// Modify the value of num by dereferencing ptr and assigning a new value
*ptr = 20; // This changes the value at the address ptr is pointing to (i.e., num becomes 20)

// Print the updated value of num
cout << "Updated value of num: " << num << endl; // Output: 20

What is Dereferencing?

Dereferencing is the process of accessing or modifying the value stored at the memory address that a pointer is pointing to. When you dereference a pointer, you use the * operator to go to the address stored in the pointer and access or change the value at that address.

int value = 42;
int *ptr = &value; // ptr now holds the address of value
int dereferencedValue = *ptr; // dereferencing ptr to get the value (42)

Types of Pointers

  • Null Pointer: Points to nothing (nullptr in C++).
  • Void Pointer: Can point to any data type but cannot be directly dereferenced.
  • Dangling Pointer: Points to memory that has been freed.
  • Wild Pointer: Uninitialized pointer with an arbitrary address.

Pointer Arithmetic

Pointers support operations like incrementing and decrementing to navigate contiguous memory, especially useful with arrays.
