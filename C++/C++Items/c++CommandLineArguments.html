Command Line

Command line arguments are optional arguments passed by the operating system to the main() function of a C++ program. Unlike normal function arguments, which are passed during the run time of a program, command line arguments are passed when the program is executed.

Passing Command Line Arguments

Command line arguments are passed at the point of execution. Suppose there is a C++ file called greeting.cpp in the current directory. It can be compiled with the following:

g++ greeting.cpp -o greeting

What Happens During Compilation
  1. Syntax Checking: The compiler checks the code for syntax errors.
  2. Translation: If there are no errors, it translates the C++ code into machine code that the computer can execute.
  3. Output: The -o greeting part specifies that the output file (the executable) should be named greeting. If you omit -o greeting, the default output would be a.out.
Next step:

The compiler has translated the program into an executable called greeting. Normally, we can run it with:

./greeting
