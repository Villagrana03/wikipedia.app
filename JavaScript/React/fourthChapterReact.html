Event Listener and Event Handlers in a Component

Your function components can include event handlers. With event handlers, we can run some code in response to interactions with the interface, such as clicking.

import React from 'react';

//Whenever we click on the button, the "onClick" function will occur, and it calls the function handleClick()
function SubmitButton() {
function handleClick() {
alert('Submission Successful.');
}
return <button onClick = {handleClick}>Submit</button>; //once the user clicks the buttonn it will call the handleClick()
}

export default SubmitButton;
