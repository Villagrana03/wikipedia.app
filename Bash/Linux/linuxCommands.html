sed

sed, short for Stream Editor, is a powerful command-line utility used for parsing and transforming text in Unix-like operating systems. It is commonly used for tasks such as:

  • Text Substitution: Replacing specific patterns in a text file with new strings.
  • Text Deletion: Removing lines or portions of text that match certain criteria.
  • Text Insertion: Adding new lines or content to specific locations within a file.
  • Text Extraction: Selecting and outputting only certain parts of a file based on patterns.
