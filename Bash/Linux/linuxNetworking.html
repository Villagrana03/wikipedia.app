Networking

Networking in Linux refers to the communication between computers and devices over a network. It involves the configuration, management, and troubleshooting of network connections, enabling devices to share resources, exchange data, and access the internet. Linux provides various command-line tools to help users manage network settings, diagnose connectivity issues, and secure communications.

ping

The ping command is used to test the reachability of a host on a network. It sends ICMP echo request packets to the target IP address and waits for replies. This helps determine if the host is reachable and measures the round-trip time for messages sent to the destinatio

ping

ping Flags

  • `-c < count >`: Stop after sending < count > packets.
  • `-i < interval >`: Wait < interval > seconds between sending packets.
  • `-t < ttl >`: Set the time to live (TTL) for the packets.
  • `-s < size >`: Specify the number of data bytes to be sent.

ifconfig

The ifconfig command is used to configure and display network interface parameters. It allows users to view current network settings, such as IP addresses and network masks, and to enable or disable network interfaces.

ifconfig

ifconfig Flags

  • `< interface >`: Display or configure the specified network interface.
  • `up`: Activate the network interface.
  • `down`: Deactivate the network interface.
  • `netmask < mask >`: Set the network mask for the interface.
  • `broadcast < address >`: Set the broadcast address for the interface.

ssh

The ssh (Secure Shell) command is used to securely connect to a remote computer over a network. It provides a command-line interface for remote administration and file transfers, encrypting all data exchanged between the client and server to ensure confidentiality and integrity.

ssh

ssh Flags

  • `< user >@< host >`: Connect to a remote host as the specified user.
  • `-p < port >`: Specify a different port to connect to on the remote host.
  • `-i < identity_file >`: Use the specified file as the private key for authentication.
  • `-v`: Enable verbose mode for debugging connection issues.
