System Information

System information in Linux refers to the data and statistics about the system's hardware and software configuration. This includes details about CPU usage, memory usage, disk space, running processes, and overall system performance.

top

This command displays a dynamic, real-time view of the systemâ€™s processes. It shows CPU and memory usage, along with a list of running processes sorted by their resource consumption. It's useful for identifying resource-hogging processes.

top

top Flags

  • `-d seconds>`: Set the delay between updates (in seconds).
  • `-u user>`: Show processes for a specific user.
  • `-p pid>`: Monitor specific processes by PID.
  • `-n number>`: Set the number of updates before exiting.

htop

Similar to top, htop provides an interactive process viewer with a more user-friendly interface. It allows users to scroll through the process list and provides color-coded information about resource usage. Users can also send signals to processes directly from the interface.

htop

htop Flags

  • `-d delay>`: Set the delay between updates.
  • `-p pid>`: Monitor specific processes by PID.
  • `--sort-key key>`: Sort processes by a specific key (e.g., CPU, memory).

df

This command displays information about disk space usage on mounted filesystems. It shows the total space, used space, available space, and the mount points for each filesystem. It's useful for monitoring disk usage.

df

df Flags

  • `-h`: Display sizes in human-readable format (e.g., GB, MB).
  • `-a`: Include dummy filesystems.
  • `-T`: Display filesystem type.
  • `-i`: Show inode information instead of block usage.

du

The du command estimates and displays the disk space used by files and directories. By default, it summarizes the disk usage of the current directory and its subdirectories. It helps users understand which files or directories are consuming the most space.

du

du Flags

  • `-h`: Display sizes in human-readable format.
  • `-s`: Show only a total for each argument.
  • `-a`: Include files as well as directories.
  • `-c`: Produce a grand total.

free

This command provides information about the system's memory usage, including total memory, used memory, free memory, and cached memory. It helps users monitor RAM usage and assess the need for memory upgrades or optimizations.

free

free Flags

  • `-h`: Display sizes in human-readable format.
  • `-m`: Show memory in megabytes.
  • `-g`: Show memory in gigabytes.
  • `-t`: Display a total line.
