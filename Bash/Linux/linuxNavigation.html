Linux Navigation

Navigation commands in Linux are a set of commands that allow users to move through the file system and manage directories and files. These commands enable you to:

  • View your current location: Determine which directory you are currently in.
  • List files and directories: See the contents of a directory.
  • Change directories: Move between different folders.
  • Create and delete directories: Manage the structure of your file system.

pwd

The pwd (print working directory) command displays the current directory you are in within the file system. It shows the full path from the root directory to your current location.

.

pwd Flags

  • `-P`: Prints the physical directory, resolving any symbolic links.
  • `--help`: Displays help information about the command.

cd

The cd (change directory) command is used to navigate between directories. You can specify the path of the directory you want to move to.

.

cd flags

  • `..`: Move up one directory level.
  • `~`: Move to the home directory.
  • `-`: Switch to the previous directory.
  • `<directory>`: Specify the target directory to navigate to.

ls

he ls command lists the contents of the current directory. It shows files and subdirectories, providing a quick overview of what is available.

.

ls flags

  • `-l`: List in long format, providing details like permissions and file sizes.
  • `-a`: Show all files, including hidden files (those starting with a dot).
  • `-h`: Human-readable sizes (used with `-l`).
  • `-R`: Recursively list subdirectories.
  • `-t`: Sort by modification time.
