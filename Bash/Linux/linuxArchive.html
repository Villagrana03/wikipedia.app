Archive Utilities

Archives are collections of files and directories that have been bundled together into a single file. This is typically done to facilitate storage, transfer, or backup. Archives can also include compression to reduce file size. Here are some key aspects of archives:.

Commands:
  • zip: Creates a zip archive, which can contain multiple files and directories.
  • tar: is a command-line utility used in Unix and Linux systems for combining multiple files and directories into a single file

have the ability to compress and archive files. This means that unlike the compression commands which compress a single file at a time, the files’ size will be reduced and packaged into a single archive file in one command.

zip

zip <archive_name>.zip <file1> <file2> …
//Example:
zip archive.zip test1.txt test2.text test3.txt //all the test will be inside of archive

//Note: On some distributions of Linux, zip must first be installed using the command sudo apt install zip unzip.

Extract files

unzip archive.zip

Zip Flags
  • -r: Recursively zip files in a directory, including subdirectories.
  • -e: Encrypt the zip file with a password.
  • -u: Update existing files in the zip archive with newer versions.
  • -d: Delete files from the zip archive.
  • -x: Exclude specific files from being zipped.
  • -v: Verbose mode; shows the files being processed.
  • -q: Quiet mode; suppresses output.
  • -o: Specify the output zip file name.

tar

it is recommended to use tar when distributing archives among Linux-based systems. This is because tar archives store Unix file attributes, retaining file permissions and other metadata.

tar -cf <archive_name>.tar <files or directories>
//Example:
tar -cf example.tar index.html script.js style.css //you are creating a tar archive named example.tar that contains the three specified files: index.html, script.js, and style.css. Here’s a breakdown of the command:

Extract files

To extract the files in a .tar archive, we can use the -xf option.

tar -xf <archive_name>.tar

tar Flags
  • -c: Create a new archive.
  • -x: Extract files from an archive.
  • -f: Specify the filename of the archive.
  • -v: Verbose mode; lists files being processed.
  • -z: Compress the archive using Gzip.
  • -j: Compress the archive using Bzip2.
  • -t: List the contents of the archive.
  • -C: Change to a directory before performing any operations.

Compressing .tar Files

What makes tar powerful is that we can combine it with the compression utilities we discussed in a previous exercise to create a compressed archive.

For example, to compress an archive/tarball of some video files video1.mp4 and video2.mkv, where we want the resulting compress archive to be called videos.tar.bz2

tar -cjf videos.tar.bz2 video1.mp4 video2.mkv

Decompressing and Extracting .tar Files

tar -xjf videos.tar.bz2
