Network Utilities

Interacting with Files Hosted Online

Let’s start simple: how do we access files from the Internet in the command line? There are two commands that allow us to do just that: curl and wget

curl: is a command-line tool used for transferring data to or from a server using various protocols, including HTTP, HTTPS, FTP, and more. It is widely used for testing and interacting with web services and APIs

$ curl "URL in quotes" //displays the contents of the file at the URL in terminal.

The option -O (a capitalized letter “O”) or -output combined with a desired name

$ curl "URL" --output <desired_filename>
//Example:
curl --output filename.html "https://www.google.com"

curl flags
  • -X <command>: Specifies the request method to use (e.g., GET, POST, PUT, DELETE).
  • -d : Sends data in a POST request (use with -X POST).
  • -H : Adds custom headers to the request (e.g., <code>-H "Authorization: Bearer token"</code>).
  • -O: Saves the output to a file named after the remote file.
  • -o <filename>: Saves the output to a specified filename.
  • -L: Follows redirects (useful when a URL redirects to another).
  • -i: Includes the HTTP response headers in the output.
  • -s: Silent mode; does not show progress or error messages.
  • -v: Verbose mode; displays detailed information about the request and response.
  • -u <user:password>: Sends basic authentication credentials.
  • -k: Ignores SSL certificate validation (useful for testing with self-signed certificates).
  • -F: Sends form data (useful for file uploads).
  • --data-urlencode : URL-encodes the specified data before sending.
  • --header: Adds a custom header (same as -H).
  • -A <agent>: Sets the User-Agent string.
  • --cookie <cookie>: Sends specified cookies in the request.

Checking Network Connectivity

How can we check whether two devices connected to the same network can communicate with one another? Simple! We use a command called ping, short for Packet INternet Groper. The command

sends packets to the target host and waits for replies. If the ping command returns with a failure, that means the target is unreachable.

ping [options] <target domain or IP>

wget

wget is a command-line utility used for downloading files from the web. Here are some basic usage examples:

wget [options] [URL]

wget flags
  • -O - Save the file with a specific name.
  • -b - Download in the background.
  • -c - Continue an interrupted download.
  • --limit-rate - Limit the download speed.
  • --recursive - Download files recursively.
  • --no-parent - Prevent downloading files from parent directories.
  • -i - Read URLs from a file.
  • --user-agent - Specify a custom user-agent string.
  • --quiet - Turn off output messages.
  • --no-check-certificate - Ignore SSL certificate checks.

DNS Resolution

What if we wanted to find out the IP address for any given domain name or vice versa? This process is known as DNS lookup or DNS resolution

Host command

host <domain or IP>

Network Interface Status

The ifconfig command, or interface configurator, is one of the network inspection commands available on Linux. Running it will display IP addresses, MAC addresses, and some more relevant details.

ifconfig [interface] [options]
