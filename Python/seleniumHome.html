Welcome to Selenium

Selenium with Python is used to carry out automated test cases for browsers or web applications. You can easily use it to simulate tests such as tapping on a button, entering content to the structures, skimming the entire site

This is the setup of selenium in firefox in my computer

from selenium import webdriver
from selenium.webdriver.firefox.service import Service

# Set up the geckodriver path
service = Service("C:\\geckodriver-v0.35.0-win32\\geckodriver.exe") # This is the path in my computer!
driver = webdriver.Firefox(service=service)

# Example: Open a webpage
driver.get("https://www.google.com")

# Close the browser
driver.quit()
