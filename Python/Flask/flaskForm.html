FlaskForm Class

A FlaskForm class comes from Flask-WTF, which is an extension of Flask that integrates Flask with WTForms. WTForms is a library that helps you manage and validate forms in Flask applications.

FlaskForm form attributes

  • StringField - Text input field
  • PasswordField - Password input (masked text)
  • TextAreaField - Multiline text input
  • EmailField - Email input field
  • DecimalField - Input for decimal numbers
  • IntegerField - Input for integer numbers
  • FloatField - Input for floating-point numbers
  • BooleanField - Checkbox for True/False
  • FileField - File upload input
  • RadioField - Group of radio buttons
  • SelectField - Dropdown select menu
  • SelectMultipleField - Multi-select dropdown
  • SubmitField - Submit button
  • DateField - Input for date (YYYY-MM-DD)
  • DateTimeField - Input for date and time
  • TimeField - Input for time only
  • URLField - Input for URLs
  • TelField - Input for telephone numbers
  • HiddenField - Hidden input field
  • SearchField - Search input field
  • DecimalRangeField - Slider for decimals (HTML5)
  • IntegerRangeField - Slider for integers (HTML5)
