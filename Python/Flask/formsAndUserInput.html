Forms and User Input

HTML forms are used to gather user input, like text, numbers, selections, or files. A form is created using the < form > tag, and it typically contains various input fields (like textboxes, radio buttons, checkboxes, etc.). The form can be submitted to the server, which processes the data.

Basic HTML form Example

< form action="/submit" method="POST">
--- < label for="name">Name:
--- < input type="text" id="name" name="name">< br>< br>

--- < label for="email">Email:
--- < input type="email" id="email" name="email">< br>< br>

--- < button type="submit">Submit< /button>
< /form>

Handling Form Submissions (GET vs POST)

When a user submits a form, the data is sent to the server. The two most common methods for sending data from a form are GET and POST.

GET Method:

  • Sends form data in the URL (as query parameters).
  • Suitable for non-sensitive data like search queries.
  • The data is visible in the browserâ€™s address bar.

#GET Method

< form action="/search" method="GET">
--- < input type="text" name="query"> #If the user submits the form with the value "python", the browser will navigate to /search?query=python, and the data will be part of the URL.
--- < button type="submit">Search< /button>
< /form>

POST Method:

  • Sends form data in the body of the HTTP request (not visible in the URL)
  • Suitable for sensitive data like passwords or large amounts of data.
  • The data is not visible in the browser's address bar.

#POST Method

< form action="/submit" method="POST">
--- < input type="text" name="username">
--- < button type="submit">Submit< /button>
< /form>

Retrieving Form Data with request.form

After the form is submitted, Flask allows you to retrieve the data submitted through the form using request.form. This is a dictionary-like object that stores the form data as key-value pairs.

Example:

#HTML File

< form action="/submit" method="POST">
--- < label for="username">Username:< /label>
--- < input type="text" id="username" name="username">< br>< br>

--- < label for="password">Password:< /label>
--- < input type="password" id="password" name="password">< br>< br>

--- < button type="submit">Submit< /button>
< /form>

#app.py

@app.route("/")
def home():
username = request.form["username"] #request.form lets you retrieve value entered in the input from the name="" attribute
password = request.form["password"]
--- return
