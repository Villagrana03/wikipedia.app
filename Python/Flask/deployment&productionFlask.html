Deployment & Production

In development mode, Flask provides a built-in server and tools for debugging. It allows for hot-reloading, meaning the server automatically restarts when you make changes to the code.

Development mode

In development mode, Flask provides a built-in server and tools for debugging. It allows for hot-reloading, meaning the server automatically restarts when you make changes to the code.

export FLASK_ENV=development
flask run

Production Mode

In production mode, Flask runs without the built-in debugger and hot-reloading. It's more secure and suitable for live applications.

export FLASK_ENV=production
flask run

Gunicorn

Gunicorn (Green Unicorn) is a Python WSGI HTTP Server for UNIX. It is a pre-fork worker model that works well with web frameworks like Flask.

It is recommended for running Flask apps in production because it is more robust and can handle more concurrent requests than the built-in Flask server.

pip install gunicorn
gunicorn -w 4 -b 0.0.0.0:8000 yourapp:app

  • -w 4: Specifies the number of worker processes (adjust based on your server's capability).
  • -b 0.0.0.0:8000: Binds the server to all available IP addresses on port 8000.
  • yourapp:app: Refers to the Flask application instance (replace yourapp with the name of your module and app with your Flask app variable).
