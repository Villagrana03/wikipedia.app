SQL Functions

SQL functions are built-in commands in SQL that perform specific operations on data within a database. They can be used in SQL queries to perform calculations, manipulate strings, format dates, and more.

Aggregate Functions

COUNT()

Counts the number of rows.

SUM()

Adds up numeric values.

AVG()

Calculates the average of numeric values.

MIN()

Returns the smallest value.

MAX()

Returns the largest value.

String Functions

CONCAT()

Combines multiple strings into one.

UPPER() / LOWER()

Converts strings to uppercase or lowercase.

LENGTH()

Returns the length of a string.

Date Functions

NOW()

Returns the current date and time.

DATE()

Extracts the date part of a date-time value.

Mathematical Functions

ROUND()

Rounds a number to a specified number of decimal places.

ABS()

Returns the absolute value.
